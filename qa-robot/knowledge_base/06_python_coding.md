# Python 编程 & LeetCode

## Python 基础面试题
- **可变与不可变**: 列表/字典可变，元组/字符串/数字不可变
- **浅拷贝 vs 深拷贝**: copy.copy() vs copy.deepcopy()
- **装饰器**: @decorator，在不修改原函数前提下增加功能
- **生成器**: yield 关键字，惰性求值，节省内存
- **上下文管理器**: with 语句，__enter__ 和 __exit__

## Python 常用数据结构复杂度
- **列表**: 索引 O(1)，末尾插入 O(1)，中间插入 O(n)
- **字典**: 平均 O(1) 查找/插入/删除
- **集合**: 同字典，去重和成员检测
- **堆 (heapq)**: push/pop O(log n)
- **双端队列 (collections.deque)**: 两端插入/删除 O(1)

## LeetCode 高频题型

### 数组/字符串
- 两数之和: 哈希表 O(n)
- 最长子串: 滑动窗口 O(n)
- 合并区间: 排序 + 遍历
- 旋转数组: 三次反转

### 链表
- 反转链表: 迭代/递归
- 环形链表: 快慢指针
- 合并有序链表: 双指针
- LRU 缓存: 哈希表 + 双向链表

### 树
- 二叉树遍历: 前序/中序/后序/层序
- 最大深度: DFS/BFS
- 验证 BST: 中序遍历递增
- 最近公共祖先: 递归

### 动态规划
- 爬楼梯: dp[i] = dp[i-1] + dp[i-2]
- 最长回文子串: 中心扩展
- 编辑距离: 二维 DP
- 背包问题: 0/1 背包和完全背包

## 时间复杂度速查
- O(1): 哈希表查找、数组索引
- O(log n): 二分查找、平衡树操作
- O(n): 线性遍历
- O(n log n): 排序 (快排/归并)
- O(n²): 双层循环、简单 DP
- O(2^n): 暴力递归

## 排序算法对比
| 算法 | 平均 | 最坏 | 空间 | 稳定 |
|------|------|------|------|------|
| 快排 | O(n log n) | O(n²) | O(log n) | 不稳 |
| 归并 | O(n log n) | O(n log n) | O(n) | 稳定 |
| 堆排 | O(n log n) | O(n log n) | O(1) | 不稳 |
| 插入 | O(n²) | O(n²) | O(1) | 稳定 |

## 面试常考算法思想
- **双指针**: 有序数组的 two sum、链表找环
- **滑动窗口**: 子串/子数组问题
- **二分查找**: 有序数组查找
- **回溯**: 全排列、组合、N 皇后
- **贪心**: 区间调度、哈夫曼编码
- **拓扑排序**: 课程表、依赖解析
