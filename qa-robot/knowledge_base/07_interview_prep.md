# 面试准备 & 简历

## 简历撰写要点
- **STAR 法则**: Situation, Task, Action, Result — 每个项目这样写
- **量化成果**: 不用"提升了性能"，用"准确率 91.7%，比 baseline 提升 5%"
- **技术栈清晰**: Python, PyTorch, Transformers, Scikit-learn
- **GitHub 链接**: 代码放 GitHub，README 写清楚
- **每个项目含**: 背景 + 方法 + 成果 + 技术点

## 项目描述模板
```
项目: 中文情感分析系统
• 基于 PyTorch + TextCNN 实现，4 种尺寸卷积核提取 n-gram 特征
• 数据预处理、词表构建、训练评估全流程
• 测试准确率 91.7%，支持交互式预测
• GitHub: github.com/fourting587/nlp-ai
```

```
面试官好，我是XX大学计算机专业大三学生。
我主要做 NLP/深度学习方向，会用 PyTorch 搭建和训练模型。

```

## 面试流程
1. **自我介绍** (1-2 min): 简洁、突出亮点
2. **项目深挖** (10-15 min): 技术细节、难点、为什么这么选
3. **算法题** (20-30 min): LeetCode medium，白板写代码
4. **基础知识** (10-15 min): ML/DL/NLP 理论
5. **反问环节** (5 min): 问团队技术栈、业务方向

## 高频面试问题

### 深度学习
- 为什么 ReLU 比 Sigmoid 好？
- 梯度消失怎么解决？
- BatchNorm 的原理？
- Dropout 训练和推理时的区别？

### NLP
- 为什么 Transformer 比 RNN 好？
- 注意力机制的公式？
- BERT 和 GPT 的区别？
- 什么是位置编码？

### 项目相关
- 为什么用 TextCNN 而不是 BERT？
- 怎么处理数据不平衡？
- 怎么评估模型好坏？
- 如果数据不够怎么办？

## 反问环节好问题
- 团队目前的技术栈是什么？
- 实习生主要做什么方向的工作？
- 团队有多少人？怎么协作？
- 有没有转正机会？
