# 深度学习基础

## 什么是深度学习？
深度学习是机器学习的一个子集，使用多层神经网络自动从数据中学习特征表示。
核心思想是通过多层非线性变换，逐层提取越来越抽象的特征。

## 神经网络的基本组成
- **输入层**: 接收原始数据
- **隐藏层**: 中间计算层，可以有多个
- **输出层**: 输出预测结果
- **激活函数**: 引入非线性（ReLU, Sigmoid, Tanh, GELU）
- **损失函数**: 衡量预测与真实值的差距
- **优化器**: 更新网络参数（SGD, Adam, AdamW）

## 常见激活函数
- **ReLU** (Rectified Linear Unit): f(x) = max(0, x)，解决梯度消失，计算快
- **Sigmoid**: f(x) = 1/(1+e^(-x))，输出范围(0,1)，适合二分类
- **Tanh**: f(x) = (e^x-e^(-x))/(e^x+e^(-x))，输出范围(-1,1)
- **GELU**: 结合了Dropout和ReLU的思想，BERT中使用
- **Softmax**: 多分类输出层，将logits转为概率分布

## 梯度下降算法
- **Batch GD**: 使用全部数据计算梯度，准确但慢
- **SGD**: 每次用一个样本，快但不稳定
- **Mini-batch GD**: 每次用一个batch，折中方案
- **Momentum**: 累积历史梯度方向，加速收敛
- **Adam**: 自适应学习率 + Momentum，最常用

## 过拟合与欠拟合
**过拟合**: 模型在训练集表现好，测试集差
- 解决方法: 增加数据、正则化(L1/L2)、Dropout、早停、数据增强

**欠拟合**: 模型在训练集上就表现差
- 解决方法: 增加模型复杂度、训练更久、减少正则化

## 正则化技术
- **L1 正则化**: 损失函数加权重绝对值之和，产生稀疏解
- **L2 正则化**: 损失函数加权重平方和，防止权重过大
- **Dropout**: 训练时随机丢弃部分神经元，相当于集成学习
- **Batch Normalization**: 每层输入归一化，加速训练
- **Layer Normalization**: 每个样本自身归一化，Transformer中使用

## 梯度消失与梯度爆炸
- **梯度消失**: 深层网络梯度趋近0，参数不更新（sigmoid易导致）
  - 解决: ReLU、残差连接、合理的初始化
- **梯度爆炸**: 梯度过大，参数震荡
  - 解决: 梯度裁剪、权重正则化

## 常见优化器
- SGD: 基础随机梯度下降
- SGD + Momentum: 加速收敛
- AdaGrad: 自适应学习率
- RMSProp: 改进AdaGrad，解决学习率单调递减
- Adam: RMSProp + Momentum，最推荐
- AdamW: Adam + 解耦权重衰减

## 训练流程
1. 数据准备（收集、清洗、划分）
2. 模型定义
3. 损失函数和优化器选择
4. 循环训练: 前向传播 → 计算损失 → 反向传播 → 更新参数
5. 验证集评估
6. 调参迭代
7. 测试集最终评估
